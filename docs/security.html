Security

Locked down by design.

Colony assumes the worst — hostile websites, poisoned repositories, compromised dependencies — so you never have to think about it.

Never leaves your machine

The server binds to localhost only — invisible to your network. No cloud, no accounts, no telemetry, ever.

A key for every request

A locally generated 256-bit token guards the API and the live socket, compared in constant time. Malicious websites scripting localhost get nothing.

No shell. Ever.

Agents cannot execute commands or reach the network. A prompt injection hidden in code can waste words — never run programs or exfiltrate data.

Path-jailed writing

Builders can only touch the one folder you gave their team — symlinks resolved, escapes blocked. Credential folders are denied outright.

Secrets stay secret

Environment files, private keys and credentials are refused at the tool level — even when an agent is explicitly asked to read them.

Hardened and audited

Strict Content-Security-Policy, rate limiting, request size caps, zero known vulnerable dependencies, and a public threat model.
