Get started

Up and running in two minutes.

You need Node 18+, pnpm, and a Claude account — either the Claude Code app signed in on this machine, or an API key. Nothing else to sign up for.

1

Get the code

Clone the repository and install dependencies.

git clone https://github.com/LEXES7/Colony.git && cd Colony && pnpm install
2

Start Colony

Run it in your browser, or as its own desktop app. The terminal prints a private link with your personal access key.

pnpm dev  pnpm desktop
3

Point it at your work

Choose the folder where your projects live, add a project, and flip it on. Its resident expert learns the codebase for you.

4

Hire a team and talk

Create a team, then just say what you want — "let's build an online store." The team plans, asks for your approval, builds, reviews, tests and audits. You watch it all happen in the office.

That's the whole setup.

Everything runs on your machine, under your control, with your account.
