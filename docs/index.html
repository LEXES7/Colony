Watch your company actually work.

Every project and team gets a room in a living office. Monitors glow while agents think, keyboards clatter, and envelopes fly between rooms when they talk. In 3D — or crisp HD pixel art.

Real tasks. Real ETAs. Real receipts.

Work is broken into estimated tasks, spread across the team, reviewed piece by piece, and traced through testing and a security audit. Live countdowns show what’s on time — and what took 8 minutes against a 10-minute estimate.

Locked down by design.

Colony assumes the worst so you don’t have to think about it.

🔌

Never leaves your machine

The server binds to localhost only. No cloud, no telemetry, no accounts. Your code and conversations stay home.

🔑

Token on every request

A locally generated 256-bit key guards the API and live socket — malicious websites can’t touch it.

🚷

Agents can’t run commands

No shell. No network tools. A prompt injection can waste words — never execute code or exfiltrate data.

📁

Path-jailed writing

Builders can only touch the one folder you gave their team. Credential folders are blocked outright.

🙈

Secrets stay secret

.env files, keys and credentials are refused at the tool level — even if an agent is asked to read them.

🧾

Hardened & audited

Strict CSP, rate limiting, zero known vulnerable dependencies, and a public threat model in SECURITY.md.
